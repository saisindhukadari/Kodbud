# Java Project Series (Task 1–8)

This repository contains a collection of Java projects I built as part of my learning journey.  
The goal was to understand Core Java concepts by implementing them through hands-on projects.

## 📌 Projects Included

### 1. Simple Calculator (CLI)
- Performs addition, subtraction, multiplication, and division
- Uses user input and switch-case logic

**Tech Used:** Java, Scanner, switch-case

### 2. Student Grade Management System
- Calculates average marks and assigns grades (A/B/C/Fail)
- Displays student report

**Tech Used:** Java, Classes, Arrays, if-else

### 3. ATM Simulation
- Menu-driven program with deposit, withdraw, and balance check

**Tech Used:** Java, Loops, Methods, Conditions

### 4. Number Guessing Game
- Difficulty levels:
  - Easy: 1–50 (10 attempts)
  - Medium: 1–100 (7 attempts)
  - Hard: 1–200 (5 attempts)
- Provides hints and tracks attempts

**Tech Used:** Java, Random, Conditions

### 5. Library Management System
- Add, display, search, and remove books
- Implements basic CRUD operations

**Tech Used:** Java, ArrayList, OOP

### 6. Simple Login System
- Username and password validation
- Displays success or failure message

**Tech Used:** Java, HashMap, String handling

### 7. Digital Clock
- Displays real-time time (HH:MM:SS AM/PM)
- Updates every second

**Tech Used:** Java, Thread, Date, SimpleDateFormat

### 8. Notepad GUI (Java Swing)
- Basic text editor with open and save functionality
- Menu-driven interface

**Tech Used:** Java Swing, File Handling, Event Listeners

## 💡 What I Learned
- Strong foundation in Core Java
- Object-Oriented Programming (OOP)
- Collections like ArrayList and HashMap
- Multithreading concepts
- Basic GUI development using Swing
- Debugging and problem-solving skills

## 🚀 About This Repository
This project series focuses on learning by building.  
Each project helped me improve my coding skills and understand real-world programming concepts.
